# Seattle Tenant-Landlord Compliance Authority (STLCA) – Housing Data

This repository contains the verified, citywide dataset powering STLCA’s housing oversight work.  
All data originates from the Seattle Department of Construction & Inspections (SDCI) public portal and has been validated, cleaned, and time-bucketed for accuracy.

### Contents
- `RowLevel_STLCA_AuditReady.csv` – Complete complaint dataset 2003–2025  
- `RowLevel_With_PhraseFlags_AND_Registration.csv` – Master flagged dataset (habitability, eviction, registration)  
- `BLANKS_2015FW_with_full_registration.csv` – Subset for unresolved or unclassified complaints  
- `Rental_Property_Registration_*.csv` – Raw registration records for cross-verification
